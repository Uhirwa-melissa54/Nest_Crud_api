# NestJS Simple CRUD REST API

## Overview
A simple NestJS REST API demonstrating CRUD operations (Create, Read, Update, Delete).  
Designed for learning and easy modification.

---

## Features
- Create, read, update, delete resources
- RESTful API endpoints
- Uses DTOs for input validation
- Minimal and easy-to-understand structure

---

## Tech Stack
- Backend: NestJS
- Language: TypeScript
- Database: In-memory or any simple database (SQLite, PostgreSQL)
- Validation: class-validator
